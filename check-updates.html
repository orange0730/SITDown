1. 首頁標籤點選檢查

問題：標籤無法點選或拖曳

解決方案：已調整 z-index 層級

如果仍有問題，在控制台執行：

// 強制修復標籤層級
document.querySelectorAll('.draggable-tag').forEach(tag => {
    tag.style.zIndex = '999';
    tag.style.pointerEvents = 'all';
    tag.style.position = 'absolute';
});
